# 贷前准入策略设计

## 策略目标

模型分数只是中间产物，最终目标是形成可执行的贷前准入策略。

策略分三档：

1. `approve`: 自动通过。
2. `manual_review`: 人工复核、补充材料、降额、提高首付或加强验证。
3. `decline`: 拒绝或进入强风控。

## 策略阈值示例

初版阈值不直接固定，而用分位数策略探索：

- 风险分数 Top 5%: decline 候选。
- 风险分数 Top 5%-20%: manual review 候选。
- 其余: approve 候选。

后续根据坏样本捕获率、误拒率和人工审核产能调整。

## 策略评估表

每组输出：

- 申请数。
- 申请占比。
- 坏样本数。
- 坏样本率。
- 坏样本捕获率。
- 相对整体坏样本率 Lift。
- 建议动作。

## 面试讲法

不要只说“模型 AUC 是多少”，而要说：

> 我把模型输出转成准入策略。比如如果人工审核产能只能覆盖申请量的 10%，我会看 Top 10% 风险组能捕获多少坏样本、命中率相比整体坏样本率提升多少、同时有多少好客户会被误拒或进入复核。这样模型结果可以直接服务审批、复核和拒绝阈值，而不是停留在算法指标。
